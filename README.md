# LunaMAIMinecraftMod

This repository contains the necessary files to connect LUNAMAI with Minecraft.

IMPORTANT:  
This project does NOT run automatically. You must set it up manually.

---

## What’s included
- Python script for the bot
- Minecraft mod file (.jar)

---

## Requirements
- Visual Studio Code  
- Python (installed)  
- Python extension for VS Code  
- Minecraft Java 1.20.1 Forge  

---

## Setup Instructions

1. Install Minecraft Forge 1.20.1  
2. Put the mod `.jar` file into your Minecraft `mods` folder  
3. Open the Python file in Visual Studio Code  
4. Install required Python libraries (in the indicated file)  
5. Run the Python script manually  

---

## How to use

- Start Minecraft with Forge  
- Run the Python script  
- Make sure both are running at the same time  

---

## Notes
- This repo is only for downloading files  
- The bot will NOT work unless the Python script is running  
- Do not close the script while using the mod
